Who I Am

Software Engineer in the medical device industry focusing on automated testing GUI applications and testing infrastructure. My education background is in computer science with a focus in information security.

Active contributor to multiple Python open source projects. If you use Python chances are very good you're running some code that I've written.

I'm also very interested in continuous integration tooling and infrastructure. I'm working on a side-project called Delt that aims to provide easy-to-read snapshots and diffs for continuous integration builds and tests.

Besides software I enjoy reading, learning, playing games of all varieties, new experiences, laughing, and pointing out relevant XKCDs.
